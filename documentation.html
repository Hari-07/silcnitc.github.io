Documentation

At the beginner level, it is required to have a good understanding of the LEX and YACC tools, inorder to develop an efficient compiler. Here, you may download the documentation related to developing the intial stages of the compiler using the tools.

Using LEX

LEX is a tool used to generate a lexical analyzer. This document covers all aspects regarding the use of LEX for SILC development. Technically, LEX is a compiler which compiles a LEX program (prog_file.l) and generates a C file (lex.yy.c) which is in turn compiled using a C compiler to generate an executable file, which is the generated lexical analyzer.
