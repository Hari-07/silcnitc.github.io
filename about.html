ABOUT THE PROJECT

Philosophy

The primary focus of the project is to make the learning process of writing a compiler an easy task. This project aims at providing provide a strong foundation of basic knowledge in compiler development which would be useful in other areas of software engineering as well.Being instructional in nature, this project gives the learner an insight into the working of LEX, YACC and the usage of these tools to develop a compiler for SIL (Simple Integer language). The theory involved would be learned on a need-to-do basis. This, in our opinion would further enhance the understanding of the theory behind the implementation process.
